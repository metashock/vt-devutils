[vt] Dev Utils

helpful functions and utilities for OXID developers

Download .zip Download .tar.gz View on GitHub

functions:

  • clearing particular caches or all cache at once
  • displaying active controller class and its main template for easy finding files and templates
  • you can keep your basket after last checkout step and easily re-send order confirmation email
  • all the debugging options and cache clearing functions can be accessed from shop frontend and backend
  • displaying last entries for shop's exception log and webserver error log
  • a scratchpad for testing some php and shop functions, you can access the whole shop framework for quick testing
  • resetting caches for some module data like included files, tempaltes, blocks or module settigns

installation:

getting the module

  • download the zip
  • extract it
  • copy the contents of the "copy_this" folder into the shop root directory

  • if you use a ftp client, make sure you set it to use binaty mode to transfer files

  • if you are familar with git and shell you can checkout the module branch:

    $ cd *your_shop_root*/modules/
$ git clone -b module git://github.com/vanilla-thunder/vt-devutils.git

activation and configuration

  • log into the OXID backend and got to extensions->modules
  • activate "vt DevUtils
  • switch to the settings tab and configure the logs. you can set absolute or relative path
  • now press F5 and you will see a new symbol in bottom left corner
  • you will have the same interface in shop frontend if you are logged in with an administrator account
